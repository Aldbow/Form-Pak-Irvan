SURVEI PENILAIAN PRIORITAS AKAR MASALAH

PENGADAAN BARANG/JASA KEMENTERIAN KETENAGAKERJAAN
Metode USG (Urgency — Seriousness — Growth)

A. Pengantar dan Petunjuk Pengisian

Yth. Bapak/Ibu Responden,
Survei ini dilaksanakan dalam rangka penyusunan Rancangan Aksi Perubahan (RAP) PKA pada UKPBJ Kementerian Ketenagakerjaan. Tujuannya adalah mengidentifikasi dan memprioritaskan akar masalah tata kelola pengadaan barang/jasa (PBJ) berdasarkan penilaian para pemangku kepentingan terkait.

Kriteria Penilaian:
U (Urgency): Tingkat mendesak masalah untuk segera diselesaikan.
S (Seriousness): Tingkat keparahan dampak jika masalah tidak ditangani.
G (Growth): Tingkat pertumbuhan masalah jika dibiarkan.

Skala Penilaian (1-5):
  • 1 = Sangat Rendah (Tidak mendesak / tidak berdampak / tidak berkembang)
  • 2 = Rendah (Kurang mendesak / kecil dampak / lambat berkembang)
  • 3 = Sedang (Cukup mendesak / cukup berdampak / cukup berkembang)
  • 4 = Tinggi (Mendesak / berdampak nyata / berkembang signifikan)
  • 5 = Sangat Tinggi (Sangat mendesak / sangat berdampak / berkembang pesat)
